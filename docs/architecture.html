⑦ Auth Flows
⟳ Page Load / Silent Refresh
1
App mounts → POST /auth/refresh (httpOnly cookie)
2
✓ Success → setAccessToken(token, user) → isHydrating=false → RoleRedirect
3
✗ Fail (no backend) → clearAuth() → isHydrating=false → AuthGuard → /login
4
While isHydrating=true → AuthGuard renders Spinner
🔑 Login Flow
1
LoginForm submit → useLogin() mutation → authApi.login() (mock, in-memory)
2
Validate email+password → makeToken() → return accessToken + user
3
mutation.onSuccess → setAccessToken() → shared store updated (all MFEs see it)
4
handleSuccess → navigate(/admin) for admin · navigate(/dashboard) for seeker
5
AuthGuard passes (isAuthenticated=true) → RBACGuard checks permission → MFE renders
